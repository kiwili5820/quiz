# /new-wordsearch — 초성 찾기 퀴즈 앱 생성

워드크로스 격자에서 초성만 보고 단어를 찾는 퀴즈를 새로 만들 때 사용한다.
각 단계마다 사용자 확인 후 다음으로 넘어간다.

---

## Step 1. 기본 정보 확인

사용자에게 다음을 질문한다:

1. **과목 폴더명** — 예: `math_A`
2. **단원 번호** — 예: `1` → `chap1` 폴더 생성

---

## Step 2. 이미지 분석

사용자가 퀴즈판 이미지를 제공하면 아래 순서로 분석한다:

1. 격자 크기(가로×세로 셀 수) 파악
2. 격자 내 초성 배치 위치 파악
3. 찾아야 할 단어 목록 및 정답 파악
4. 단어별 격자 내 위치(시작 좌표, 방향) 파악

분석 결과를 아래 형식으로 정리해 사용자에게 보여주고 **확인을 받은 후** Step 3으로 넘어간다.

```
격자 크기: N×M
단어 목록:
  "단어" | 초성: "ㅊ ㄱ" | 위치: (row, col) | 방향: 가로/세로
  ...
```

---

## Step 3. index.html 생성

확인된 데이터로 `{과목폴더}/chap{N}/index.html`을 생성한다.

- 해당 과목의 CLAUDE.md 규칙을 따른다
- `APP_URL = "https://kiwili5820.github.io/quiz/{과목폴더}/chap{N}/index.html"`
- 이미지 경로는 `../img/` 기준으로 작성

생성 후 사용자에게 검토를 요청한다.

---

## Step 4. 커밋 & 푸시

사용자 승인 후 실행한다:

```
git add {과목폴더}/chap{N}/
git commit -m "feat: {과목폴더} chap{N} 초성 찾기 퀴즈 추가"
git push origin master
```

GitHub Pages 반영까지 1~2분 소요됨을 안내한다.
