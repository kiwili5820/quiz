# sports_S — 초성 찾기 유형 규칙

## 퀴즈 유형

**워드서치 격자에서 초성만 보고 단어를 찾는 퀴즈**  
격자에 초성이 표시되어 있고, 학생이 해당 단어를 입력하거나 선택한다.
단어마다 힌트 버튼을 제공하여 필요 시 힌트를 확인할 수 있다.

---

## 퀴즈 데이터 구조

```js
const PUZZLE = {
  cols: N,
  rows: M,
  words: [
    {
      id: 1,
      answer: "운동전이",
      chosung: "ㅇㄷㅈㅇ",
      hint: "한 운동 기술을 배우면 다른 운동 기술 학습에 영향을 미치는 현상",
      row: 0, col: 0,
      direction: "across"   // "across" | "down"
    },
    // ...
  ]
};
```

---

## hint.md 형식

```
초성 >> 단어
힌트 텍스트

초성 >> 단어
힌트 텍스트
```

이미지 분석 후 hint.md를 읽어 각 단어의 `hint` 필드를 채운다.

---

## 힌트 버튼 UI

- 단어 목록 각 항목 오른쪽에 `<img src="../img/tip.png">` 버튼 배치
- 버튼 클릭 시 해당 단어의 힌트 텍스트를 토글 표시 (기본: 숨김)
- 힌트 텍스트는 버튼 아래 또는 옆에 작은 텍스트로 표시

---

## 이미지 경로 규칙

- `index.html` 위치: `{과목폴더}/chap{N}/`
- 힌트 아이콘: `../img/tip.png`
- 퀴즈판 이미지: `chap{N}/` 폴더 안에 위치 (같은 폴더이므로 파일명만)

---

## 이미지 분석 → 코딩 절차

1. 퀴즈판 이미지에서 격자 크기 파악
2. 격자 내 초성 배치 및 단어 위치(시작 좌표, 방향) 파악
3. hint.md를 읽어 각 단어의 초성·정답·힌트 텍스트 매핑
4. PUZZLE 구조로 데이터화 → `puzzle_data.js`로 저장 후 사용자 확인
5. 확인 후 `index.html` 생성
